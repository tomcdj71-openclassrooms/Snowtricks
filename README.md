## Pre-requisites :
- PHP 8.2
- Composer
- npm (I used pnpm)
- Symfony CLI

## Install

```sh
git clone https://github.com/tomcdj71/Snowtricks
cd Snowtricks
composer install --no-dev --optimize-autoloader
yarn install
yarn build
symfony console d:d:c
symfony console d:m:m
symfony console d:f:l
symfony serve
```
